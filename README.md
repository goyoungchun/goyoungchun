# Personal Product Studio

Placeholder-first Next.js site scaffold for a personal product studio.

## Included
- Home / About / Projects / Project Detail / Privacy / Privacy Detail / Contact
- File-based placeholder content
- Reusable components
- Basic metadata helpers

## Next Steps
1. Replace example copy in `content/` and page sections
2. Replace `siteUrl` and `contactEmail` in `content/site.ts`
3. Add real product screenshots / assets under `public/images/`
4. Connect deployment (recommended: Vercel)

## Run
```bash
npm install
npm run dev
```
